The Kimi Project

Kimi est un moyen très simple pour vous protéger des personnes malveillantes sur Discord.

Le robot possède une base de donnée appelée Blacklist. Dès qu'un utilisateur malveillant entre sur un serveur où Kimi est installé, il sera instantanèment banni, l'empêchant de refaire des actions malveillantes sur votre serveur.
