The Kimi Project

Kimi est un moyen très simple pour vous protéger des personnes malveillantes sur Discord.

Le robot possède une base de donnée appelée Blacklist. Dès qu'un utilisateur malveillant entre sur un serveur où Kimi est installé, il sera instantanèment banni, l'empêchant de refaire des actions malveillantes sur votre serveur.

Robot Discord - Le Guide

Le guide Robot Discord sert à vous apprendre les bases du développement de robots Discord (ne me remerciez pas, c'est naturel) en vous faisant progresser pas-à-pas très doucement, le tout dans une mise en page exemplaire pour un confort de lecture inégal. Un guide pour toute la famille !
